OpenAI Vision（寻款，推荐）

填写 API Key 后，拍照寻款优先使用 gpt-4o-mini 看图+语义匹配库内「编号+特征」；按次计费、无月费。导入表格时可选择是否为每行参考图自动生成一句视觉描述（会按行调用 API）。

火山方舟 · 豆包视觉（寻款）

国内网络直连，拍照寻款走方舟 OpenAPI（兼容 OpenAI Chat Completions）。在火山控制台创建 Doubao-vision 接入点，将 Endpoint ID（形如 ep-xxxx）填入下方；API Key 填入 Access Key（Bearer）。未启用 Google 且启用本项时，优先于 OpenAI。

Google Product Search（寻款）
重要（2025 起常见）：Google 已将经典 Product Search 置于维护/受限状态。新账号或未获白名单时，接口可能返回 Product Search is in maintenance mode，属官方策略而非本系统配置错误。 可选：向 Google 申请继续使用 Product Search； 长期方案见 Vision Warehouse（图搜替代）。 在此之前请关闭本项，改用设置中的 OpenAI Vision 或 阿里云图搜。

启用后，拍照寻款优先走 Google Cloud Vision 商品图搜；导入 CSV/Excel 时会将参考图写入 GCS 并注册到指定 Product Set。类目默认 homegoods-v2（饰品场景）。首次请在 GCP 控制台开通 Vision API，并执行 php scripts/google_create_product_set.php 创建 ProductSet。

密钥 JSON 必须放在项目根目录之外的绝对路径（不可位于 public 下），切勿提交到 Git 或暴露 URL。

导入时会上传 gs://bucket/...，索引完成后对象可保留或自行配置生命周期。
阿里云图像搜索（寻款）

用于「拍照寻款」以图搜款；导入 CSV/Excel 时会将图片同步到阿里云商品图搜实例。AK/SK 建议使用 RAM 子账号最小权限。

官方类目表见阿里云文档「图像搜索图片类目与ID列表」。
