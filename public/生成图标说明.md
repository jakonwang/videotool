# PWA 图标生成说明

PWA Builder 需要以下图标文件：

## 需要的图标文件

1. **favicon.ico** - 网站图标（已存在或需要创建）
2. **icon-192.png** - 192x192 像素的 PNG 图标
3. **icon-512.png** - 512x512 像素的 PNG 图标

## 快速生成图标的方法

### 方法一：使用在线工具（推荐）

1. **访问图标生成器**
   - https://realfavicongenerator.net/
   - 或 https://www.favicon-generator.org/

2. **上传你的图标**
   - 准备一个至少 512x512 的图片
   - 上传到工具

3. **下载生成的图标**
   - 下载所有尺寸的图标
   - 将 `android-chrome-192x192.png` 重命名为 `icon-192.png`
   - 将 `android-chrome-512x512.png` 重命名为 `icon-512.png`
   - 放到 `public` 目录下

### 方法二：使用 PWA Builder 自动生成

PWA Builder 可以自动生成图标：
1. 访问 https://www.pwabuilder.com/
2. 输入你的网站地址
3. 点击 "Generate Icons"
4. 下载生成的图标包
5. 将图标放到 `public` 目录

### 方法三：手动创建（简单图标）

如果你想要一个简单的图标，可以使用在线工具：
- https://www.canva.com/ - 设计图标
- https://www.remove.bg/ - 去除背景
- 导出为 192x192 和 512x512 的 PNG

## 临时解决方案

如果暂时没有图标，可以：
1. 使用 PWA Builder 的默认图标
2. 或者创建一个简单的纯色图标

## 图标要求

- **格式**: PNG
- **尺寸**: 
  - icon-192.png: 192x192 像素
  - icon-512.png: 512x512 像素
- **背景**: 透明或纯色
- **内容**: 应用 Logo 或相关图标

## 放置位置

所有图标文件应该放在 `public` 目录下：
```
public/
  ├── manifest.json
  ├── icon-192.png
  ├── icon-512.png
  └── favicon.ico
```
